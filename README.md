# cs2dsrv
Custom CS2D Dedicated Server written in Lua

## Requirements
1. LuaJIT
2. LuaSocket

Debian-based Linux Distributions
```bash
apt install luajit lua-socket
```
or you can get [luapower](https://luapower.com/)
